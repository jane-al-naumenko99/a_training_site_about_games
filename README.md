# A training site about games — учебный многостраничный сайт

Учебный проект по вёрстке: многостраничный сайт в тематике игр/геймдева.

## Стек
- HTML
- CSS (Flexbox, Grid)
- Базовый JavaScript (простая валидация)

## Страницы
- `index.html` — главная страница (hero, trending, features, projects, подписка)
- `contact.html` — страница контактов

## Что реализовано
- Навигация и общие блоки (header/footer)
- Сетки на Flexbox и CSS Grid (в т.ч. блок подписки и футер)
- Hover‑эффекты, переходы и анимации (`transition`, `transform`)
- Подсветка активного пункта меню через `::after`
- Простая проверка email‑поля в блоке подписки (проверка `@` и `.`)

## Как запустить
Проект статический:
1. Скачайте/клонируйте репозиторий
2. Откройте `index.html` в браузере

## Структура
- `index.html`, `contact.html`
- `css/main.css` — стили
- `img/` — изображения
